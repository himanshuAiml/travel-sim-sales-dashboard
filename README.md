# 📊 Travel SIM Sales Dashboard

Welcome to my **Travel SIM Sales Dashboard** project! This dashboard is built using **HTML, CSS, JavaScript, Chart.js, and Supabase REST API** to visualize sales data, monitor business performance, and manage users, products, orders, and destinations.

---

# 📌 About

This project was developed as part of my **Summer Bootcamp 2026**. It demonstrates frontend development, REST API integration, database connectivity with Supabase, and interactive dashboard design.

The dashboard provides a clean and responsive interface for viewing real-time sales analytics and business insights.

---

# 🚀 Features

- 📈 Interactive Dashboard
- 💰 Revenue Analytics
- 📦 Product Management
- 🛒 Order Management
- 👥 User Management
- 🌍 Destination Management
- 📊 Dynamic Charts
